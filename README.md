# iPhone用の簡単なToDoアプリ

swift/swiftUIを使ってCRUD機能の備わったToDoアプリを作りました。  
Coredataを使ってデータの管理を行いました。

## Create
右上にある「＋」ボタンで新規追加

## Read
ToDoリストの項目をタップで内容を確認

## Update
Todoリストの内容を右上にある保存ボタンを押すと更新

※保存ボタンを押さないと保存されない

## Delete
ToDoリストの左上にある「Edit」ボタン、もしくは項目を左にスワイプで「Delete」を押すと削除
